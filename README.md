## 食用方法

#### 进入文件详细页面，点击右上角“下载”即可。
#### 如果没有出现当天录播的话，那么说明: 
* 正在上传中，请稍后再来 
* 该主播没有加入每日录播行列，可以在「取得联系」板块联系我添加 
* 负责上传的机器人睡着了，私信我叫我把他打醒


## 界面功能

* 点击文件上方属性可根据不同属性对文件进行排序
* 网站右上角🔍图标可快速搜索文件
* 进入文件详细页面，可点击右上角“下载”下载该文件
* 文件详细界面可使用在线播放器播放视频，也可通过下方按钮调用本地播放器播放
* 右下角🌏图标可主动切换语言
* 右下角☀图标可切换浅色/深色模式
